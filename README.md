# Пример прокидывания куки через корс

### Установка 
```
npm i
```

### Запуск клиента

```
npm start:client
```

Открыть в браузере localhost:3000

### Запуск сервера

```
npm start:server
```


Считаем что сервер запущен на 127.0.0.1:3001, кука будет устанавливаться на этот домен.
Запросы будут уходить тоже на этот адрес.
